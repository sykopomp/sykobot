Sykobot - An IRC bot from another universe!

About

Sykobot is a next-gen AI, programmed in a next-gen language. Project goals include an extensible, object-oriented modular architecture, a flexible yet easy-to-use commands and plugin system, and world domination.

Features

-search support
-parrot function
-facts
-grabs
-kiloseconds
-memos
-command aliases
